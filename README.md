# 🎂 Bring Your Own Cake - Event Website

A fun, mobile-optimized website for the "Bring Your Own Cake" picnic event in Boise, Idaho, built with Astro and Tailwind CSS.

## ✨ Features

- **Live Countdown Timer** - Counts down to September 20th, 2025
- **Interactive Ticket Button** - Fun animation when clicked (currently just for show)
- **Event Description** - Clear explanation of what the event is about
- **Social Media Links** - Ready for your social media accounts
- **Mobile-First Design** - Optimized for all device sizes
- **Cutesy Theme** - Fun colors and animations perfect for a cake event

## 🚀 Getting Started

### Prerequisites

- Node.js (version 18.20.8 or higher recommended)
- npm

### Installation

1. Clone or download this project
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:4321`

## 🎨 Customization

### Colors

The website uses custom cake-themed colors defined in `tailwind.config.mjs`:

- `cake-pink` - Soft pink
- `cake-yellow` - Light yellow
- `cake-blue` - Sky blue
- `cake-purple` - Light purple
- `cake-orange` - Peach orange

### Event Details

Update the event information in `src/pages/index.astro`:

- Event date and time
- Location details
- Social media links
- Event description

## 📱 Mobile Optimization
